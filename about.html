About Project

ShopEase Pro is a modern e-commerce website developed as a final year project. It includes features like login system, cart, checkout, admin panel, and database view.


This project is built using HTML, CSS, and JavaScript and can be connected to PHP/MySQL backend.

Features

  • ✔ User Login & Signup
  • ✔ Product Search & Filter
  • ✔ Shopping Cart
  • ✔ Checkout System
  • ✔ Admin Panel
  • ✔ Dark Mode
